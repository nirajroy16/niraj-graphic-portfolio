# Niraj Roy Graphic Portfolio

Personal portfolio website for Niraj Roy, a Graphic Designer and Brand Identity Specialist based in Guwahati, Assam.

Live site: https://niraj-graphic-portfolio-nirajr436-3816s-projects.vercel.app

## Overview

This portfolio presents Niraj's packaging, label, brand identity, and product visual work across FMCG, tea, spices, bakery, dairy, and home-care categories. The site is a lightweight static website built with plain HTML, CSS, and JavaScript for fast deployment and easy maintenance.

## Highlights

- Responsive portfolio layout
- Product work gallery with packaging-focused presentation
- Full-time experience at Manjushree Plastics and freelance brand work
- Resume download
- Certificate link
- Contact links for email, phone, Instagram, LinkedIn, and Facebook
- Deployed on Vercel

## Tech Stack

- HTML
- CSS
- JavaScript
- Vercel

## Project Structure

```text
.
```

The `public/` folder contains the Vercel-ready static output and site assets.

## Deployment

The project is deployed on Vercel. For updates:

```bash
git add .
git commit -m "Update portfolio"
git push origin main
vercel --prod
```
